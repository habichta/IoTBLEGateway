Detailed Description

Representation of a BlueGiga BLED112 Device.

Implements relevant functions to manage/control a Bluegiga BLED112 Device in the context of a gateway application The implementation follows the specifications of BGAPI to communicate with the BLED112 over a TCP connection. The basic architecture looks as follows:
